 # 🚀 CodeGuard AI

> AI-Powered Code Review & Security Analysis Platform

CodeGuard AI is an intelligent code review platform that helps developers identify bugs, security vulnerabilities, code smells, and performance issues using advanced Large Language Models (LLMs).

The platform provides automated code analysis, AI-generated review comments, security recommendations, and actionable improvement suggestions to accelerate software development and improve code quality.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/codeguard-ai.git
cd codeguard-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

Create `.env.local`

```env
DATABASE_URL=
OPENAI_API_KEY=
GEMINI_API_KEY=
ANTHROPIC_API_KEY=
JWT_SECRET=
```

### Setup Database

```bash
npm run db:generate
npm run db:push
```

### Start Development Server

```bash
npm run dev
```

Open:

http://localhost:3000
